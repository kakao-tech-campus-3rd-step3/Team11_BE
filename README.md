# Team11_BE

## 프로젝트 한 줄 소개 ✨

- 즉석/근거리 번개 모임을 빠르게 만들고 참여하는 프로젝트의 백엔드 설계 및 구현


## 구조 🧩
- Backend
  - database: PostgreSQL + MongoDB
  - cache/broker:  Redis
  - server: Spring Boot


## 핵심 목표 🎯
- 외부 로그인(OAuth2) + JWT
- 위치 기반 모임 검색/생성
- 실시간 채팅(WebSocket/STOMP)

## 백엔드 기술 스택 🛠️
- Java 21, Spring Boot 3
- Spring Security + OAuth2
- JPA + PostgreSQL(+PostGIS)
- MongoDB(채팅 히스토리)
- Redis(Cache/Pub-Sub)
- WebSocket + STOMP

## 진행 상태 🚧
- 초기 설계 단계 — 상세 문서는 구현 진행에 따라 추가 예정
